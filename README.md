#Shazam Clone (Audio Fingerprinting Project)

This project is an educational implementation of an audio fingerprinting system inspired by Shazam.

##Features

- FFT-based spectral analysis
- Magnitude conversion to decibel scale
- Frequency domain visualization
- Modular project structure

##Tech Stack

- Python 3.11
- NumPy
- SciPy
- Librosa
- Matplotlib
